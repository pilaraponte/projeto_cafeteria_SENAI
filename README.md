


## Requisitos
- Node.js 18+ (Windows)
- NPM

## Como rodar

### Backend
```bat
cd backend
npm install
npm run db:push
npm run seed
npm start
```
API: http://localhost:3001

### Frontend
```bat
cd frontend
npm install
npm run dev
```
App: http://localhost:5173
